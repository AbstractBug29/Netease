1.Project Name:简陋的网易云歌单分析工具，Enter the user's NetEase cloud uid, and crawler analysis can be performed on the user's historical listening song rankings to obtain the user's historical playback volume ranking, most frequently listened to singers, type and style tags of listening songs, and recommend 20 songs for users based on this , and information about users with similar tastes.

2.Click on the only py file in the folder and click Run to run the program

3.Before running, you need to install BeautifulSoup4, selenium, jieba, wordcloud, pygame, matplotlib, alive_progress and numpy libraries.You also need to download nodeJS and deploy the api locally, see the NeteaseCloudMusicApi-master document for details

4.To run the program, we only need to enter the uid and click the corresponding button
